❌ Backend chưa chạy!

Để start backend, làm theo các bước sau:

Option 1: Dùng Terminal trong VS Code

  1. Mở VS Code
  2. Mở Terminal: Ctrl + `
  3. Chạy lệnh:
cd backend
npm run dev

Option 2: Double-click file

Mở file start-backend.bat trong thư mục dự án

Option 3: Dùng PowerShell Script

.\start-backend-local.ps1

Sau khi start xong, click nút "Kiểm tra lại" bên dưới.
