Test Menu Navigation

Đây là trang test để kiểm tra menu navigation hoạt động đúng trên cả desktop và mobile.

Hướng dẫn test:

  • Desktop: Hover vào các mục "Giới thiệu" và "Dịch vụ" để xem dropdown menu
  • Mobile: Click vào nút hamburger menu để mở/đóng menu
  • Mobile Dropdown: Click vào "Giới thiệu" hoặc "Dịch vụ" để mở dropdown
  • Thoát menu: Click ra ngoài hoặc nhấn ESC để đóng menu

Các tính năng đã cải thiện:

  • ✅ Hamburger menu button với hiệu ứng biến thành dấu X
  • ✅ Menu mobile full screen với overlay
  • ✅ Dropdown menu cho "Giới thiệu" và "Dịch vụ"
  • ✅ Hiệu ứng smooth và animation
  • ✅ Khóa scroll body khi menu mở
  • ✅ Đóng menu khi click ra ngoài hoặc nhấn ESC
  • ✅ Responsive design cho mọi kích thước màn hình
