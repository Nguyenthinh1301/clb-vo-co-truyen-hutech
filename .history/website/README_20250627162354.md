# ThongTin-VCT Website

Chào mừng đến với dự án website ThongTin-VCT! Đây là một trang web hiện đại và responsive được xây dựng với HTML, CSS và JavaScript.

## Cấu trúc dự án

```
website/
├── index.html          # File HTML chính
├── styles.css          # File CSS cho giao diện
├── script.js           # File JavaScript cho chức năng
├── assets/             # Thư mục chứa hình ảnh, icon
│   ├── images/
│   ├── icons/
│   └── videos/
├── docs/               # Tài liệu hướng dẫn
└── README.md           # File hướng dẫn này
```

## Tính năng

- **Thiết kế responsive**: Tương thích với mọi thiết bị (desktop, tablet, mobile)
- **Navigation mượt mà**: Smooth scrolling giữa các section
- **Giao diện hiện đại**: Sử dụng gradient, shadow và animation
- **Tối ưu hóa**: Code sạch sẽ và được tối ưu hóa cho hiệu suất

## Cách sử dụng

### 1. Mở website
- Mở file `index.html` trong trình duyệt web
- Hoặc sử dụng Live Server extension trong VS Code để phát triển

### 2. Tùy chỉnh nội dung
- Chỉnh sửa file `index.html` để thay đổi nội dung
- Cập nhật file `styles.css` để thay đổi giao diện
- Thêm tính năng mới trong file `script.js`

### 3. Thêm assets
- Đặt hình ảnh vào thư mục `assets/images/`
- Đặt icon vào thư mục `assets/icons/`
- Đặt video vào thư mục `assets/videos/`

## Cấu trúc HTML

Website bao gồm các section chính:
- **Header**: Navigation menu cố định
- **Hero**: Section chào mừng với call-to-action
- **About**: Phần giới thiệu
- **Services**: Danh sách dịch vụ
- **Contact**: Thông tin liên hệ
- **Footer**: Thông tin bản quyền

## Hướng dẫn phát triển

### Thêm section mới
1. Thêm HTML structure trong `index.html`
2. Thêm styles tương ứng trong `styles.css`
3. Cập nhật navigation menu nếu cần
4. Thêm JavaScript functionality nếu cần

### Tùy chỉnh màu sắc
Các màu chính được sử dụng:
- Primary: #3498db (xanh dương)
- Secondary: #2c3e50 (xanh đậm)
- Accent: #e74c3c (đỏ)
- Background: #f8f9fa (xám nhạt)

### Responsive breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Performance Tips

1. Tối ưu hóa hình ảnh trước khi upload
2. Sử dụng WebP format cho hình ảnh nếu có thể
3. Minify CSS và JavaScript cho production
4. Sử dụng CDN cho external libraries

## Deployment

### GitHub Pages
1. Push code lên GitHub repository
2. Enable GitHub Pages trong Settings
3. Chọn source branch (usually main/master)

### Netlify
1. Connect GitHub repository
2. Set build command (không cần nếu là static site)
3. Set publish directory thành "."

### Vercel
1. Import project từ GitHub
2. Deploy automatically
